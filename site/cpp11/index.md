# C++11 — The Modern Era

**Year:** 2011 · **Concepts:** 213 (53 core + 160 STL)

C++11 was the biggest update to C++ since its creation. It introduced modern programming idioms that transformed how C++ is written, focusing on safety, expressiveness, and performance.

## Sections

- [**Core Language** (53 concepts)](/cpp11/core) — Move semantics, lambdas, `auto`, range-for, `nullptr`, variadic templates, and more.
- [**STL & Library** (160 concepts)](/cpp11/stl) — Smart pointers, `std::thread`, `std::atomic`, `unordered_map`, `std::tuple`, and the full modern library.

## Key Features

- **Move semantics** — rvalue references, `std::move`, move constructors
- **Lambdas** — anonymous functions with capture lists
- **`auto` and type deduction** — less boilerplate, more clarity
- **Smart pointers** — `unique_ptr`, `shared_ptr`, `weak_ptr`
- **Range-based for** — `for (auto& x : container)`
- **`nullptr`** — replaces `NULL`
- **Variadic templates** — type-safe variadic functions
- **`std::thread`** — portable multi-threading
- **`constexpr`** — compile-time evaluation
- **Initializer lists** — uniform initialization syntax
- **`std::array`**, **`std::unordered_map`**, **`std::tuple`**
