# C++98 — The Foundation

**Year:** 1998 · **Concepts:** 213 (52 core + 161 STL)

C++98 is the original ISO standardization of C++, establishing the language fundamentals, the Standard Template Library (STL), and the object-oriented and generic programming model that all later standards build upon.

## Sections

- [**Core Language** (52 concepts)](/cpp98/core) — Classes, templates, inheritance, RAII, exceptions, and all fundamental language features.
- [**STL & Library** (161 concepts)](/cpp98/stl) — Containers, algorithms, iterators, strings, streams, and the full standard library.

## Key Features

- Classes, constructors, destructors, RAII
- Templates (function templates, class templates)
- Standard Template Library (vector, map, list, set, …)
- Exceptions (`try` / `catch` / `throw`)
- Namespaces, operator overloading
- `const` correctness, references, pointers
- Iterators and generic algorithms (`sort`, `find`, `transform`, …)
- I/O streams (`cin`, `cout`, file streams)
- Inheritance, virtual functions, polymorphism
